## ✨ Features

- 🌑 **Dark Glassmorphism UI** with animated gradient orbs
- ⌨️ **Typewriter animation** cycling through roles
- 📊 **Animated CGPA & stats counters** on scroll
- 🃏 **3D tilt effect** on project cards
- 📈 **Skill proficiency bars** that animate on scroll-into-view
- 🖱️ **Cursor glow** that follows the mouse
- 🔗 **Active nav highlight** as you scroll through sections
- 📬 **Contact form** with animated submit feedback
- 📱 **Fully responsive** — mobile, tablet & desktop
- ⚡ **Zero frameworks** — pure HTML, CSS & Vanilla JS
